# Imdb-movies-data-analysis
Business Problem

The film industry invests significant resources in producing movies, but predicting audience reception before release is challenging. Understanding which factors contribute to a movie's success helps production companies make informed decisions.

This project analyzes historical IMDb movie data to identify the key factors influencing movie popularity and ratings. By examining features such as genre, budget, cast, director, duration, Facebook popularity, and user engagement, the project develops a predictive model capable of estimating a movie's IMDb rating.

The insights obtained from this analysis can help filmmakers, producers, and entertainment companies better understand audience preferences and make more informed production and marketing decisions
