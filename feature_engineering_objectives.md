# FEATURE ENGINEERING OBJECTIVES

1.  To create new meaningful features such as Profit, ROI, Movie Age,
    and Genre Count that provide additional information beyond the
    original dataset.

2.  To transform existing variables into more informative
    representations, including logarithmic transformation of user votes
    to reduce skewness.

3.  To combine related features, such as Facebook likes and review
    counts, into aggregate metrics that better represent movie
    popularity and audience engagement.

4.  To convert continuous variables into categorical features, such as
    Budget Category and Rating Category, making the data easier to
    analyze and suitable for classification if required.

5.  To improve the quality and predictive capability of the dataset by
    engineering relevant features that enhance machine learning model
    performance.
